# 2026-04-10 플랭크 시간 기반 채점 규칙

## 목표

플랭크를 `SIDE` 전용 시간 기반 운동으로 지원한다.

- 자유 운동: 세션 시작 전 목표 시간 직접 설정
- 루틴 운동: `target_value`를 목표 시간(초)으로 사용
- 세션 결과: 최고 연속 hold 시간 저장
- 최종 점수: 자세 80 + 시간 20

## 상태 정의

- `SETUP`: 측면 자세로 진입했지만 아직 안정화 확인 전
- `HOLD`: 연속 안정 프레임 조건을 충족한 유효 hold 구간
- `BREAK`: 정렬 붕괴, 프레임 이탈, 품질 저하로 hold가 끊긴 상태

## 시간 규칙

- 사용자 타이머는 `SETUP` 진입 즉시 시작한다.
- 공식 결과값은 `best_hold_sec`(세션 내 최고 연속 hold 시간)만 사용한다.
- `BREAK` 이후 다시 자세를 잡아도 이전 hold와 이어 붙이지 않는다.

## 점수 규칙

- `posture_score`: 플랭크 최고 hold 구간의 평균 자세 점수
- `time_score = min(100, (best_hold_sec / target_sec) * 100)`
- `final_score = posture_score * 0.8 + time_score * 0.2`

## 핵심 메트릭

- `hip_angle`: 몸통 일직선
- `spine_angle`: 척추 안정성
- `shoulder_angle`: 어깨 지지 정렬
- `knee_angle`: 다리 펴기

## UI 원칙

- 자유 운동 플랭크는 세션 페이지에서 목표 시간 설정 후 시작한다.
- 세션 중에는 아래 값을 동시에 보여준다.
  - 현재 구간 시간
  - 최고 기록
  - 목표 시간
  - 현재 자세 점수
  - 현재 상태(`SETUP`/`HOLD`/`BREAK`)
- 목표 시간 도달 시 자동 종료하지 않는다.
